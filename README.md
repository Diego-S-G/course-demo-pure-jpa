# Mini Projeto JPA Puro com Java e MySQL (course-demo-pure-jpa)
<br/>

Este é um mini projeto que fiz acompanhando o [curso de Java do Nelio Alves na Udemy](https://www.udemy.com/course/java-curso-completo).
<br/>

Basicamente é uma demonstração do uso do JPA (Jakarta Persistence API) puro com Java e MySQL, utilizando Hibernate como implementação.

O projeto mostra como:
- Mapear entidades com JPA
- Configurar o arquivo `persistence.xml`
- Realizar operações básicas de persistência, consulta e remoção de dados usando o `EntityManager`
- Gerenciar manualmente o `EntityManagerFactory` e o `EntityManager` (sem Spring)

Ideal para quem quer entender o funcionamento do JPA puro antes de partir para frameworks como Spring Boot.
